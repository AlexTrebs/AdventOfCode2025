# RULES

The instructions for this day to find the maximum amount of rolls of paper (marked `@`) a forklift can get.

A warehouse can look like this:

```
..@@.@@@@.
@@@.@.@.@@
@@@@@.@.@@
@.@@@@..@.
@@.@@@@.@@
.@@@@@@@.@
.@.@.@.@@@
@.@@@.@@@@
.@@@@@@@@.
@.@.@@@.@.
```

For part one we are told that a roll of paper can only be taken my a forklift if there is less than 4 rolls of paper adjecent to it (including diagonally adjecent).

So for the warehouse above we can mark the avaliable `13` rolls of paper with `X` to get:

```
..xx.xx@x.
x@@.@.@.@@
@@@@@.x.@@
@.@@@@..@.
x@.@@@@.@x
.@@@@@@@.@
.@.@.@.@@@
x.@@@.@@@@
.@@@@@@@@.
x.x.@@@.x.
```

For part two there is a recursiveness to the method. this means if a roll is removed, there is potential to remove more rolls.
