# RULES

Our task is to fix a machine, that machine has a diagram showing how it is layed out.

The machince has lasers which can only move down, the diagram looks like this:

```
.......S.......
...............
.......^.......
...............
......^.^......
...............
.....^.^.^.....
...............
....^.^...^....

...^.^...^.^...
...............
..^...^.....^..
...............
.^.^.^.^.^...^.
...............
```

```
.......S.......
.......|.......
......|^|......
......|.|......

.|^|||^||.||^|.
.|.|||.||.||.|.
|^|^|^|^|^|||^|
|.|.|.|.|.|||.|
```

For part one we would like the output of the number of times the beam is split which is `21`.

For part two we want to find how many total paths there are for the laser, so for istance for the exmaple:
